Epik
=======================
Get the whole story.

### What is it?
Epik is a client-server application that helps you collect rich data from users.

### Installation
```bash
# install with git
git clone https://github.com/jordanpatton/epik-server.git
cd epik-server
npm install
bower install
```

### Usage
```bash
node app.js
```
